# 🐾 Veterinary Chatbot API

API backend para un chatbot de clínica veterinaria enfocado en gestión de esterilización y logística clínica.

---

## 🚀 Deploy

API en producción:

👉 https://clinica-veterinaria-crmn.vercel.app/api

---

## 🧠 Descripción del proyecto

Esta API proporciona un chatbot inteligente diseñado para asistir a usuarios en tareas relacionadas con una clínica veterinaria, especialmente en:

- Esterilización de mascotas
- Preparación preoperatoria
- Información logística (horarios, citas, entrega de animales)
- Orientación general para procesos clínicos

El sistema utiliza inteligencia artificial (OpenAI) para generar respuestas dinámicas adaptadas al contexto del usuario.

---

## ⚠️ Limitaciones del sistema

Por motivos de seguridad y responsabilidad:

- ❌ No realiza diagnósticos médicos
- ❌ No prescribe tratamientos
- ❌ No sustituye a un veterinario

✔️ Solo proporciona información general y orientación logística
